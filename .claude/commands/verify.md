## Petlog 핵심 사용자 흐름

변경 후 아래 흐름이 정상 동작하는지 순서대로 확인한다.

### 1. 인증 흐름
- [ ] 회원가입 → 이메일/비밀번호 입력 → 성공
- [ ] 로그인 → JWT 토큰 발급 → 대시보드 이동
- [ ] 로그아웃 → 토큰 제거 → 로그인 화면 이동
- [ ] 인증 없이 보호된 페이지 접근 → 로그인 화면 리다이렉트

### 2. 반려동물 등록 흐름
- [ ] 반려동물 등록 폼 접근
- [ ] 필수 정보 입력 (이름, 종류, 품종, 생년월일)
- [ ] 등록 완료 → 반려동물 목록에 표시

### 3. 건강 기록 흐름
- [ ] 반려동물 선택 → 건강 기록 탭 접근
- [ ] 건강 기록 작성 (체중 / 식사 / 증상 등)
- [ ] 기록 목록에서 시간 순 확인
- [ ] 빈 상태(Empty state) 표시 확인

### 4. 에러 / 비동기 상태
- [ ] API 호출 중 로딩 UI 표시
- [ ] 필수 입력값 누락 시 유효성 에러 메시지
- [ ] 네트워크 에러 시 에러 메시지 표시

### 5. 모바일 UX
- [ ] 375px(iPhone SE) 기준 레이아웃 깨짐 없음
- [ ] 터치 타겟 크기 적절 (최소 44x44px)
