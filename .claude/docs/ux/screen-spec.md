# Petlog 화면 명세

## 1. 화면 목록

| 화면 | 경로 | 설명 |
|------|------|------|
| 로그인 | /login | 이메일 로그인 |
| 회원가입 | /signup | 신규 사용자 가입 |
| 홈 | / | 반려동물 대시보드 |
| 반려동물 등록 | /pets/new | 반려동물 정보 입력 |
| 반려동물 상세 | /pets/:petId | 반려동물 건강 현황 |
| 건강 기록 추가 | /pets/:petId/records/new | 건강 기록 입력 |
| 건강 타임라인 | /pets/:petId/timeline | 기간별 건강 변화 |
| 병원 기록 | /pets/:petId/medical | 병원 방문 기록 |
| 약 관리 | /pets/:petId/medications | 복용 중인 약 |
| AI 리포트 | /pets/:petId/reports | AI 건강 리포트 |

---

## 2. 로그인 화면 (/login)

### 목적

기존 사용자 인증.

### 주요 요소

- 이메일 입력 필드
- 비밀번호 입력 필드
- 로그인 버튼
- 회원가입 링크

### 상태

- 로딩: 로그인 버튼 비활성화 + 스피너
- 에러: 이메일 또는 비밀번호 오류 메시지
- 성공: 홈 화면으로 이동

---

## 3. 홈 화면 (/)

### 목적

반려동물 목록과 빠른 기록 진입점 제공.

### 주요 요소

- 반려동물 목록 카드
- 각 반려동물의 최근 건강 상태 요약
- 기록 추가 FAB (Floating Action Button)
- 반려동물 추가 버튼

### 상태

- 로딩: 스켈레톤 UI
- 반려동물 없음: 등록 유도 빈 상태
- 정상: 반려동물 카드 목록

---

## 4. 반려동물 상세 화면 (/pets/:petId)

### 목적

선택한 반려동물의 건강 현황 요약.

### 주요 요소

- 반려동물 프로필 (이름, 종류, 나이)
- 최근 건강 기록 요약
- 오늘의 기록 현황
- 빠른 기록 버튼
- 탭: 타임라인 / 병원 / 리포트

---

## 5. 건강 기록 추가 화면 (/pets/:petId/records/new)

### 목적

빠르고 간편한 건강 기록 입력.

### 주요 요소

- 기록 종류 선택 (몸무게 / 식사 / 활동량 / 증상)
- 값 입력 필드 (종류에 따라 변경)
- 메모 입력 필드 (선택)
- 기록 날짜 / 시간 (기본값: 현재)
- 저장 버튼

### 기록 종류별 입력 형태

몸무게:

- 숫자 입력 + 단위 (kg)

식사:

- 선택지 (잘 먹음 / 보통 / 적게 먹음 / 안 먹음)

활동량:

- 선택지 (활발함 / 보통 / 적음 / 거의 없음)

증상:

- 텍스트 입력

---

## 6. 건강 타임라인 화면 (/pets/:petId/timeline)

### 목적

기간별 건강 변화 시각화.

### 주요 요소

- 기간 필터 (1주 / 1개월 / 3개월)
- 기록 종류 필터
- 변화 그래프 (몸무게 등 수치 데이터)
- 날짜별 기록 목록

### 상태

- 기록 없음: 빈 상태 메시지 + 기록 추가 유도

---

## 7. AI 리포트 화면 (/pets/:petId/reports)

### 목적

AI가 분석한 건강 리포트 확인.

### 주요 요소

- 리포트 생성 버튼
- 기간 선택 (주간 / 월간)
- 생성된 리포트 목록
- 리포트 상세 내용
  - 건강 변화 요약
  - 주목할 기록
  - 보호자 안내 메시지

### 상태

- 생성 중: 로딩 애니메이션 + 안내 문구
- 리포트 없음: 생성 유도
- 생성 실패: 에러 메시지 + 재시도 버튼

---

## 8. 모바일 레이아웃 원칙

모든 화면은 모바일 우선으로 설계한다.

기준:

- 최대 너비: 390px (iPhone 기준)
- 하단 네비게이션 바 고정
- FAB는 우측 하단 고정
- 스크롤은 수직 단방향

### 하단 네비게이션

| 탭 | 아이콘 | 이동 경로 |
|----|--------|-----------|
| 홈 | 집 모양 | / |
| 타임라인 | 그래프 | /pets/:petId/timeline |
| 기록 추가 | + | /pets/:petId/records/new |
| 병원 | 병원 | /pets/:petId/medical |
| 리포트 | 문서 | /pets/:petId/reports |
