# Petlog Project Context

## Goal

Petlog는 풀스택/프로덕트 엔지니어 이직을 위한 포트폴리오 프로젝트이다.

단순 CRUD 구현이 목적이 아니다.

실제 서비스 개발 과정에서 필요한:

- 사용자 문제 정의
- 제품 설계
- 도메인 모델링
- 확장 가능한 아키텍처
- AI 기능 확장 구조

를 보여주는 것이 목표다.


## MVP Goal

웹 우선 모바일 경험 제공.

핵심 사용자 흐름:

1. 사용자 가입
2. 반려동물 등록
3. 건강 기록 작성
4. 건강 변화 확인
5. AI 기반 건강 요약 확인


## Development Scope

현재:

- Mock AI
- PostgreSQL
- NestJS API
- Next.js Frontend

Later:

- 실제 LLM 연동
- 모바일 앱 출시
- 글로벌 서비스 확장


## Non Goal

초기 버전에서 하지 않는다.

- 의료 진단
- 수의사 대체
- 복잡한 커뮤니티
- 과도한 AI 기능
